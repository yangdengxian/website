## Usage

* First,clone git repo and cd to project folder:

```
	$ git clone https://github.com/viseye/website.git
	$ cd website/ 
```

* Then,install required node packages:

```
	$ npm install 
```

* And finally, start the app:

```
	$ npm start
```
or
```
	$ node app.js
```

open `http://localhost:18081` with browser.
